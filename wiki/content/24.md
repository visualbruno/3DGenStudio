# Texturing

Using a ComfyUI workflow, you can draw on the mesh using a mask.

![image](http://localhost:3001/wiki-media/1780049062231-67425219.png)

You have some parameters you can change about your mask
![image](http://localhost:3001/wiki-media/1780049145051-137459604.png)


In this example, I draw a mask on the wood and ask ComfyUI with this prompt:
> change the look of the wood, make it full of cracks

![image](http://localhost:3001/wiki-media/1780049290125-958875454.png)

Result

![image](http://localhost:3001/wiki-media/1780049389127-646875269.png)

If you are satisfied with the result, click on **Apply** button

You can then overwrite the mesh or save it as a new version
![image](http://localhost:3001/wiki-media/1780049514415-717157044.png)

If you save it as a **New version**, it will be visible as an "Edit" in the Assets page.
![image](http://localhost:3001/wiki-media/1780049572322-497683733.png)
