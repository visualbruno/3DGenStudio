# Imports / Exports

3DGenStudio can move work in and out of the app in two ways:

- **[Project](37.md)** — export an entire project (with all its assets) to a self-contained `.3dgp` bundle, and import one back. Use this to back up a project, move it to another machine, or share it with a teammate.
- **[Mesh](38.md)** — export a single mesh to a 3D file format (GLB, FBX, OBJ, PLY, STL) with engine-ready presets for Blender, Unity and Unreal. Use this to take a finished asset into a game engine or DCC tool.

Pick the page that matches what you want to move:

| I want to… | Use |
| --- | --- |
| Back up, move, or share a whole project | **[Project import / export](37.md)** |
| Send one mesh to Blender / Unity / Unreal / another tool | **[Mesh export](38.md)** |
