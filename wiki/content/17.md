# Image Edit

Move your card from "Images" column to "Image Edit" column.

Click on the "Action" button and you can either call an external API or run a ComfyUI Workflow.
![{932188A9-B4A1-46EC-8ABB-2AA7DB161C04}](http://localhost:3001/wiki-media/1780044299317-579173823.png)

The image edited here will be added as a "child" of the selected image. It will be marked as "Edit".

![{6CED7DBD-A8C5-4436-B448-1190B1AC3981}](http://localhost:3001/wiki-media/1780044431573-110215682.png)

You can continue to edit an image until you are ready for the next step. In this example, I delight the original image and then removed the background.

**Make sure to select the correct image in the dropdown**

![{6D2BEF7D-BF24-4362-86E3-D117C89B1971}](http://localhost:3001/wiki-media/1780044526385-632801433.png)
