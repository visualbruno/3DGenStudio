# Auto UV

Automatically unwrap your mesh into a clean UV atlas — no manual seam painting required. Auto UV segments the surface into charts, parameterizes each one, and packs them into a single texture atlas.

> Auto UV runs on the **Python mesh-tools service**. Make sure it is configured and running in **Settings → Mesh Tools** before using this mode.

## Running it

1. Open the **Auto UV** mode in the left toolbar.
2. Adjust the parameters below (the defaults work well for most organic meshes).
3. Click **Run Auto UV** and wait for the service to finish.
4. The result replaces the current mesh and shows a preview of the UV layout. Click **Keep** to accept it, or **Revert** to go back to the previous mesh.

The result panel also reports metrics (chart count, distortion, etc.) so you can compare before committing.

## Parameters

### Segmentation

Controls how the surface is cut into charts before unwrapping.

| Field | Default | What it does |
| --- | --- | --- |
| **Normal-cone cap** | 50° | Maximum angular spread of face normals within one chart. Higher = fewer, larger, more distorted charts. |
| **Sharp-edge weight** | 0.35 | How strongly sharp edges attract seams. Higher pushes seams onto creases. |
| **Fold cap** | 88° | Dihedral fold angle that forces a seam. Edges that bend more than this are always cut. |
| **Min faces / chart** | 20 | Charts with fewer faces than this are dissolved into their neighbours. |
| **Min area fraction** | 0.004 | Minimum chart area as a fraction of the total surface area. Kills tiny slivers. |

### Refinement

An optional pass that merges small charts back together when merging doesn't add too much distortion.

| Field | Default | What it does |
| --- | --- | --- |
| **Validated merge pass** | On | Runs an LSCM-validated chart merge. Turn off for a faster unwrap that produces more charts. |
| **Merge below faces** | 80 | Charts smaller than this face count are candidates for merging. *(Only when the merge pass is on.)* |
| **Merge distortion cap** | 1.32 | Maximum angle-distortion ratio a merge is allowed to introduce. *(Only when the merge pass is on.)* |

### Parameterization

How each chart is flattened into 2D.

| Field | Default | What it does |
| --- | --- | --- |
| **Method** | Auto | Flattening algorithm: **Auto**, **LSCM**, **ARAP**, or **Planar**. |
| **ARAP iterations** | 4 | Number of As-Rigid-As-Possible relaxation passes. 0 disables ARAP (LSCM / planar only). |

### Packing

How the charts are arranged into the final atlas.

| Field | Default | What it does |
| --- | --- | --- |
| **Atlas resolution** | 1024 px | Target atlas size (256 → 8192 px). |
| **Padding** | 4 texels | Gutter between charts to prevent bleeding when sampling. |

### Topology repair

Cleans up broken geometry before unwrapping.

| Field | Default | What it does |
| --- | --- | --- |
| **Proximity weld** | On | Welds coincident vertices before unwrapping — stitches shattered shells back together. |
| **Weld tolerance** | 0.1 | Weld distance, as a fraction of the median edge length. *(Only when weld is on.)* |

---

The result replaces the mesh; use **Keep** or **Revert** to decide.
