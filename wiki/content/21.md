# Import a new Workflow

Click on the **Import JSON** button.

![{F546C886-B508-46B0-A87A-B1315326FFAC}](http://localhost:3001/wiki-media/1780047308760-183606128.png)

Then you can decide which parameters will be available and which one will be used for the result.

1. Outputs
In this example, **Save Image**.

**Pay attention to select the correct output type Image**

![{1C58913C-7A99-4DEF-83E7-E70694989719}](http://localhost:3001/wiki-media/1780047644283-606716519.png)

2. Inputs

Select all **Inputs** you want to be available.
Look for your input, in this example, I look for "Prompt"
![{D34DC913-46AB-4F07-B362-672915FA52A5}](http://localhost:3001/wiki-media/1780047682445-634952198.png)

![{19D7E16D-F660-4612-A750-429F81404424}](http://localhost:3001/wiki-media/1780047715486-573415478.png)

Another useful input, is the seed. Pay attention that the type must be **Number**

![{C4294AC6-5F05-46A4-AF42-C5E570868E28}](http://localhost:3001/wiki-media/1780047842445-692705095.png)

![{717492DF-DB7F-4B31-8348-28C74226A605}](http://localhost:3001/wiki-media/1780047856609-50384901.png)

Another input, the image resolution

![{0A5E2CE5-8537-418E-84AB-A5AB79956A40}](http://localhost:3001/wiki-media/1780047973175-98199087.png)

3. Save your workflow

Use it in Kanban board or Graph Page
