# Mesh Generation

Now, you are ready to generate your mesh from the selected image.

You have the same actions as "Image Edit" column, but the available ComfyUI Workflows only allow to generate a mesh.

**Pay attention to select the correct image in the dropdown**

![{6D5953DE-2E4C-4980-9F0C-F0347622241F}](http://localhost:3001/wiki-media/1780044730316-852045795.png)

Run the workflow and wait for the result.

![{A35D02A0-81DC-466B-BEB2-84F6DD4EDB8A}](http://localhost:3001/wiki-media/1780044760304-421091068.png)

You mesh is generated.
![{3F542AEE-63BF-45A2-8907-55592DE8B969}](http://localhost:3001/wiki-media/1780045157391-291859044.png)

If you click on it, it will open the **Mesh Preview Dialog**
![{ECFBC79C-BFA3-484D-855B-99AF031A90FE}](http://localhost:3001/wiki-media/1780045203461-927865697.png)

If you click on "Edit" button, it will open the **Mesh Editor**
![image](http://localhost:3001/wiki-media/1780045254515-407568194.png)
