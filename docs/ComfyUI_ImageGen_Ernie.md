### Go to the Assets page and import your workflow

<img width="1182" height="621" alt="{0CCF7F6A-A2C2-42B4-A6DB-810CDE7FD490}" src="https://github.com/user-attachments/assets/6b107ba6-b9ff-4026-875d-eb8de4260d01" />

1) Unselect all input parameters
2) The output "Save Image", make sure to select "Image" as the type of the output
<img width="191" height="357" alt="{9238B2D1-E79A-4C99-AAE9-05351018FE8E}" src="https://github.com/user-attachments/assets/51430239-b1d9-430a-a932-af535b61d4f5" />

3) Check the input parameter "Positive Prompt" and rename it "Prompt"
<img width="260" height="205" alt="{708966F7-02FD-436D-93AE-3C2DDC810CBE}" src="https://github.com/user-attachments/assets/0d51c0d2-f0b2-432e-a35e-bbbbb806c35b" />

4) Check the input parameter "KSampler Seed" and rename it "Seed"
<img width="221" height="172" alt="{EAA8A88B-BD2A-4A45-AE56-20A72D0E844A}" src="https://github.com/user-attachments/assets/17a68330-371b-436f-b055-9b88d52a840a" />

5) You can check other parameters if you need
6) Click on "Save Workflow"
